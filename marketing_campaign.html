Strategic Insights into Marketing Campaigns: iFood Case Study

Goal: To analyze and predict customer engagement with iFood's marketing campaigns using advanced machine learning techniques. The project focused on identifying the key demographic and behavioral factors that drive campaign success, with the aim of optimizing marketing strategies and enhancing customer targeting.

Key Technologies and Tools

  • Programming: Python, Pandas, NumPy
  • Data Analysis: SQL Server Analysis Services (SSAS), DAX
  • Machine Learning: Random Forest, Scikit-learn
  • Data Visualization: Matplotlib, Seaborn, PowerBI
  • Development Environment: Visual Studio, Google Colab
